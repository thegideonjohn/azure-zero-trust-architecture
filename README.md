# Project: The Zero-Trust Fortress 

A hands-on Azure project demonstrating a Zero-Trust architecture. This repository documents the entire process of building, securing, and automating a production-ready environment in Microsoft Azure.

---

## Final Architecture & Features Overview

This entire project is segmented into three logical phases, documented in the files linked below.

### The Foundation (Project 1)
* **Goal:** Build the Hub-Spoke network and deploy secure, private resources.
* **Key Skills:** **Hub-Spoke Topology**, **VNet Peering**, Private Endpoints, Managed Identity, and RBAC.

### The Enforcement (Project 2)
* **Goal:** Implement central traffic control and secure access.
* **Key Skills:** **Azure Firewall** (Basic Tier), **Azure Bastion**, **Forced Tunneling (UDRs)**.

### The Intelligence (Project 3 - Upcoming)
* **Goal:** Build the Mini-SOC for automated defense.
* **Key Skills:** **Microsoft Sentinel**, **KQL**, and **Logic Apps (SOAR)** for auto-remediation.
